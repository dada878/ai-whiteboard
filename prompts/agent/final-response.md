# 最終回應 Prompt

請基於所有收集的資訊，用自然友善的語氣回答使用者的原始問題：「{{originalQuestion}}」

## 回答要求

1. **資訊整合**：綜合所有搜尋和分析的結果
2. **誠實透明**：如果資訊不完整，請誠實說明找到了什麼，還缺什麼
3. **具體例證**：請用具體的例子和引用來支持你的答案
4. **清晰結構**：有條理地組織你的回答

## 語氣指引

- 友善且專業
- 清晰且準確
- 適度使用表情符號來增加親和力（但不要過度）
- 用繁體中文回答
